# UAVnintei

2020年の世界のドローンビジネス市場は14兆円ともいわれ、ドローンの活躍の場は、測量以外でも農林水産業・点検・物流・捜索など多岐にわたっています。今、空のドローンだけではなく点検・調査を目的に水中や陸上のドローンのニーズが高まっています。また、現在SLAM（Simultaneous Localization and Mapping）と言われている技術がありますが、SLAMとはLiDARを使い移動体の自己位置推定と環境マップを同時に作成する技術であり、マルチパスの影響が多い都市部など、GPSが使えない環境においても自動航行が可能であります。今後、地上型ドローンの普及にともない、SLAMの技術が普通に空型ドローンにも用いられることは容易に予想されます。実際、空中型ドローンにおいても”フーバーマップ”という商品名で応用されています。そのため、今後の授業展開において空中だけではなく自動制御プログラムを理解した無人機開発のエンジニアを育成することを目標にして、空のドローンだけに固執せず、普及しつつある技術を取り入れ多方面に対応できる人材を育成できるように考えています。ちなみに、現在私のほうでもROS（ミドルウェア）を用いSLAMを使った小型ロボットを動かしながら実験をしています。

今回の認定講習申請をされるにあたり、そこまで踏み込んだ授業内容は不要であるとは思いますが、近畿測量専門学校らしい特徴を出すため、近測から新しい技術をどんどん発信できるように考えています。今回はドローンに関連する自動制御技術とプログラムなどを少し盛り込んだ形でご提案をさせていただきました。
