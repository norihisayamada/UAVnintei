# UAVnintei

2020年の世界のドローンビジネス市場は14兆円ともいわれ、ドローンの活躍の場は、測量以外の農林水産業・点検・物流・捜索など多岐にわたっています。今、空のドローンだけではなく点検・調査を目的に水中や陸上のドローンのニーズが高まっています。そのため、今後の授業展開において空中だけではなく自動制御プログラムを理解した無人機開発のエンジニアを育成することを目標にしています。

今回の認定講習申請をされるにあたり、そこまで踏み込んだ授業内容は不要であるとは思いますが、新しい近畿測量専門学校らしい特徴を出すため、ドローンに関連する自動制御技術などを少し盛り込んだご提案をさせていただきました。

AI基礎（どれか１つでも授業に取り入れてみませんでしょうか）
	Python言語
	ROSを用いたロボット
	AI（機械学習）基礎実習
	IoT(Internet of Things )
	OpenCVを用いた画像処理　i-pythoを用いた画像処理入門

UAV基礎
機体のメンテナンスと飛行技術と運用
	機体の構成（バッテリ、制御装置、送受信機、機体構造）
	自動制御とプログラム（追加）
	航空力学
	UAV関連法規（電波法、航空法、PSE）
	気象とUAV操縦技術
	飛行計画作成

基礎技術
写真測量とレーザー計測（そのほかのセンシング技術）
	3次元測量　座学・実習
